# CONTROL — Olympic Mastery

> *If you cannot control the position, you do not earn the movement.*

A progressive web app for calisthenics mastery — handstand, planche, pull-ups, rings, and beyond.

**Powered by TLC · Men of Purpose OKC Foundation**

## Tech Stack

- React 18 + TypeScript
- Vite 5 (PWA-enabled)
- Tailwind CSS + shadcn/ui
- Workbox (offline-first service worker)

## Getting Started

```sh
npm install
npm run dev
```

## Install on Your Phone

1. Open the app URL in Chrome (Android) or Safari (iOS)
2. Tap **"Add to Home Screen"**
3. The app runs fullscreen like a native app — no browser bar

## Build & Deploy

```sh
npm run build
npm run preview
```
